Co je GRPC

Co jsou Protocol buffers

  • developers.google.com/protocol-buffers
  • serializace datových struktur
  • na jazyku nezávislé
message Book {
    string id = 1;
    string isbn = 2;
    string title = 3;
    Author author = 4;
}

message Author {
    string id = 1;
    string firstname = 2;
    string lastname = 3;

}
